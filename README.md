# msTodo
msTodo is a microservice plugin for the framework which exposes methods for Todo Items and lists. 
msTodo follows the CRAP schema, and it's RESTful HTTP methods are paired below.

| CRAP Verb | HTTP Verb |
|-----------|----------:|
| Create    | POST      |
| Retrieve  | GET       |
| Alter     | PATCH     |
| Purge     | DELETE    |

# Routes

/l	lists
/t	todos
/u	users
/b	board 
/z	label
